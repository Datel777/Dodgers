
3 слота робота:
1. Основной
2. Дополнительный (Второстепенный)
3. Пассивный

Модуль может вставляться в один из слотов робота, для достижения определённого эффекта

Именнованные модули узкоспециализированны и вставляются только в конкретный слот

Модули отличаются редкостью (цветом)

Редкость модулей
1. Серый - вставляется в один слот, начальные
2. Зелёный -вставляется в два слота ~(активный - 1,2 слот; пассивный - 2,3)
3. Синий - вставляется в три слота, универсальный
4. Фиолетовый - экспериментальный, работает невообразимым образом
5. Именованный - уникальный, возможна отсылка на что-то, по большей части может вставляться только в один слот

При регистрации, игроку открываются ~5 серых модулей (~из 20)
